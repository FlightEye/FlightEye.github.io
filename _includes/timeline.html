Timeline

Stay up to date about where FlightEye has been, and where we are going next!

  • Aug-Dec 2022

    Alpha Demo: Networking and Simulation

    FlightEye has demonstrated networking and simulation development. For networking, FlightEye has shown that ADS-B data can be recieved, decoded, and used on the Microsoft HoloLens. For simulation, FlightEye has shown the ability to produce a simulated environment that models the expected real-world behavior.

  • Jan-Feb 2023

    Beta-Demo: Integration

    FlightEye will demonstrate the ability to integrate real-time networking data into a simulated environment to create an accurate test bed for the predictive algorithm.

  • Mar-May 2023

    Final Demo: Full Functionality

    FlightEye will demonstrate the ability to receive real-world ADS-B data and accurately display it to the pilot wearing the Microsoft HoloLens 2.
